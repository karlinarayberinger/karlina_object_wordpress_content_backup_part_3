INTEGERS_V3
The following single page web application allows the end user to manually enter some input, n, into a text field and to click a button in order to see the factorization of n (if n is a base-ten integer no larger than 23). The application consists of the following two code files: - JavaScript logic: integers_v3.js (raw code link) - web page interface: integers_v3.html (raw code link)
