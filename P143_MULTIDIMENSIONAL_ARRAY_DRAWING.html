MULTIDIMENSIONAL_ARRAY_DRAWING

MULTI_DIMENSIONAL_ARRAY: an ordered sequence of elements such that each element represents a unique allocation of memory for storing exactly one piece of data and such that each element has a unique index number (i.e. nonnegative integer) within the array.

The drawing above depicts a multidimensional rather than a linear array (even though all multidimensional arrays “collapse” to linear arrays at the hardware level). The multidimensional array depicted in the drawing might as well be named A[2][3] because it has two rows and three columns (and three of the twelve elements is filled in with the color orange while the other nine elements are empty (and orange could represent the presence of some kind of data)).
