INTEGERS

The following simple “warm up” web page application implements a JavaScript file and an HTML5 file to manipulate countable data.

HTML5 File (integers.html)

The HTML5 file defines the web page component of the web page application. It allows the end user to input a button click. In response to a button click, the web page implements a unit test function (which updates the web page with resulting static text).

function is_exactly_one_argument(input) {
	return (arguments.length === 1) ? true : false;
}

function is_exactly_one_number_type_argument(input) {
	// Note: The following erroneous web page output is what occurs if the commented out line of code is used.
	// is_exactly_one_number_type_argument(0,0) := true. 
	// return (is_exactly_one_argument(input) && (typeof arguments[0] === "number")) ? true : false;
	return ((arguments.length === 1) && (typeof arguments[0] === "number")) ? true : false;
}

function round_down_to_nearest_integer(input) {
	if (!is_exactly_one_number_type_argument(input)) return 0;
	return Math.floor(input);
}

function round_down_to_nearest_absolute_value_integer(input) {
	if (!is_exactly_one_number_type_argument(input)) return 0;
	input = (input > 0) ? input : ((-1) * input);
	return Math.floor(input);
}

JavaScript File (integers.js)

The JavaScript file defines functions which determine whether or not exactly one discrete data value (i.e. datum) is being passed into a function as input and which convert function inputs into base-ten integer outputs.
