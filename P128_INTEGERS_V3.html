INTEGERS_V3
The following single page web application allows the end user to manually enter some input, n, into a text field and to click a button in order to see the factorization of n (if n is a base-ten integer no larger than 23). The application consists of the following two code files: - JavaScript logic: integers_v3.js (https://raw.githubusercontent.com/karlinarayberinger/karlina_object_wordpress_content_backup_part_3/main/integers_v3.js) - web page interface: integers_v3.html (https://raw.githubusercontent.com/karlinarayberinger/karlina_object_wordpress_content_backup_part_3/main/integers_v3.html)
